# EMQX Docker Deployment Guide

## Quick Start

1. Make sure you have Docker and Docker Compose installed
2. Navigate to the directory containing docker-compose.yml
3. Run the following command to start EMQX:
```bash
docker-compose up -d
```
4. Access the EMQX Dashboard:
   - URL: http://localhost:18083
   - Default credentials:
     - Username: admin
     - Password: public

To stop EMQX:
```bash
docker-compose down
```


Important Notes:
1. Configuration file is located in emqx.conf in the current directory. For configuration options, see: https://docs.emqx.com/en/emqx/v5.8.2/hocon/
2. Log directory is mounted by default in ./log
3. Data directory is mounted by default in ./data
4. Be cautious when modifying the hostname as it will reinitialize the data. The data/mnesia directory will generate a new data directory named after the hostname.

Note: This configuration is generated by https://docker.emqx.dev
